# SeamPass - Secure Password Generator

A comprehensive password generation solution with web app, browser extension, and CLI package.

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Start development
pnpm dev

# Build all packages
pnpm build
```
